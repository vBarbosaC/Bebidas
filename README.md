# Buscador de Bebidas

Aplicação web que busca bebidas usando a API TheCocktailDB.

## Tecnologias

- HTML5
- CSS3
- JavaScript ES6
- API externa

## Funcionalidades

- Buscar bebidas
- Mostrar dados da API
- Salvar no localStorage
- Excluir da tabela

## Como executar

1. Baixe o projeto
2. Abra o arquivo index.html no navegador
